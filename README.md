# astro
This module will ultimately host assorted astro-related projects.  Some
will be old modules refactored to incorporate improved documentation or
more efficient calculations.  On the other hand, some will be new projects
used to increase computational understanding.  All modules and projects
follow the Epytext documentation guidelines found at the following URL:
http://epydoc.sourceforge.net/manual-epytext.html
